# STRIVE Left or Right Basic

Basic edition of the STRIVE Independence left-right discrimination activity.

- Fixed 10-picture sessions
- Randomized from a starter set of 15 images
- Image pool includes 8 left-hand photos and 7 right-hand photos
- Includes English/Spanish interface support
- Generated documentation note remains in English

Open `index.html` directly in a browser or host the folder as a static site.
